Blackmoor Data Profiler

A local-first CSV and Excel profiling tool that helps users inspect datasets, identify common data quality issues, generate a basic data dictionary, and export Markdown profile reports.

Inspect

Upload CSV or Excel files and quickly understand row counts, columns, missing values, and sample values.

Flag Issues

Find duplicate rows, blank-heavy columns, possible IDs, possible dates, and suspicious mixed-type fields.

Document

Generate a basic data dictionary and Markdown report for lightweight dataset documentation.

Local-first by design

Blackmoor Data Profiler Free Edition is designed to run locally. Files are processed in the locally running app session and are not intentionally sent to external services.

Future Pro candidates may include AI-assisted documentation, SQL generation, dashboard readiness workflows, and richer report formats.
